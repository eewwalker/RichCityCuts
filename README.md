### RICH CITY STOPS 
An app for managing stops and users, allowing users to favorite particular stops and view their locations on a map.
* Admin can add/edit/delete/like
* User can like
* Filter stops by neighborhood
* Authentication
* MapQuest API integration
* Custom 404
